# Axis Legacy

AI-powered image transformation tools built with Next.js and React.

## Features

- Colorize black and white images
- Enhance face clarity and details
- Transform facial age characteristics
- Blend different ethnic features

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
4. Open [http://localhost:3000](http://localhost:3000)

## Tools

### Colorize Tool
Transform black and white images to color with adjustable intensity.

### Face Clarity Tool
Enhance facial details and clarity with adjustable levels.

### Age Transform Tool
Modify facial age characteristics with adjustable age changes.

### Ancestry Mix Tool
Blend different ethnic features with adjustable mix percentages.

## Technologies Used

- Next.js 13+ (App Router)
- React 18+
- TypeScript
- Tailwind CSS
- React Hook Form
- Zod for validation
